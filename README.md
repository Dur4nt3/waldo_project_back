# Where's Waldo (Backend)

## Where's Waldo (Backend) - Overview

This is the seventh project of the "NodeJS" course within "The Odin Project".

This project is used to practice the implementation of a RESTful application and testing backend routes.

In particular, this project aims to create a where's waldo like game.

*Note: This repository is for the API backend. This side of the application only includes the game's logic and data handling.*

For the client visit [this](https://github.com/Dur4nt3/waldo_project_front) repository.
